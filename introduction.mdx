## Installation

Install the package with either npm or yarn:

```bash
npm install react-native-onboard
```

or

```bash
yarn add react-native-onboard
```

## Quick start

Place the `OnboardFlow` component anywhere in your app. It will automatically
take up the entire screen. To change the default behavior, see the
[docs on supported props](/components/high-level-props).

See
[Live Demo](https://snack.expo.dev/@christian-frigade/react-native-onboard-simple-demo)
on Expo Snack.

```js
import { OnboardFlow } from "react-native-onboard";

const App = () => {
  return (
    <OnboardFlow
      pages={[
        {
          title: "Welcome to my app",
          subtitle: "This is page 1",
          imageUri: "https://frigade.com/img/example1.png",
        },
        {
          title: "Page 2 header",
          subtitle: "This is page 2",
          imageUri: "https://frigade.com/img/example2.png",
        },
      ]}
    />
  );
};
```
